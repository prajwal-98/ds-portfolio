# Project Overview - DS Portfolio (Prajwal / Maggie)

## Vision
A long-term, interview-focused data science portfolio showcasing three hero projects (ML pipeline, GenAI pipeline, Analytics dashboard) with live demos and an interactive GenAI assistant that answers questions about the author and the projects.

## Goals
- All hero projects must be live demos, not just source code.
- Interactive GenAI Q&A on the front page (backend + prompt engineering).
- Polished UI with a full-width layout: Bio → Ask Me Anything → Project slider → Quick links → Footer.
- Deploy at least one demo publicly (Render/Heroku/Netlify combo for API + UI).

## Core projects (examples)
- **Project A — ML Pipeline:** E2E pipeline, model & Streamlit demo. Tech: Python, scikit-learn, Streamlit.
- **Project B — GenAI Pipeline:** LLM-driven tagging and Q&A, FastAPI backend. Tech: Python, LLM API, FastAPI.
- **Project C — Analytics Dashboard:** interactive competitor analysis. Tech: Pandas, Plotly, Streamlit.
