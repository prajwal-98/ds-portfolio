# UI Design Plan - DS Portfolio

## Page layout (top to bottom)
1. **Hero / Bio** — name, role, short intro. Full width container with subtle border.  
2. **Ask Me Anything** — full-width GenAI chat placeholder. Later connect to FastAPI LLM endpoint.  
3. **Hero Projects Slider** — cinematic project card (one visible at a time). Thumbnail row below. Prev/Next buttons.  
4. **Quick Links** — Resume, GitHub, LinkedIn.  
5. **Footer** — copyright, contact.

## Visual rules
- Max-width container: 1100px centered with soft border and shadow.  
- Cinematic images: width 100%, height ~220px, object-fit: cover.  
- Thumbnails: 140px wide, small captions.
